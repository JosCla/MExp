L1 structure (768B total):
0-99: Various non-const static variables
    0-7: Files (EM, CE, EP, PD)
    8-9: Map width (MW)
    10-11: Map height (MH)
    12-17: Camera position (CX, CY, CZ)
    18-19: Camera draw parity (CP)
    20-23: Interaction event variables (IT, IP)
    24-99: UNUSED
100-767: Stack (S is initially L1+768)

Str1F....: File strings

GDB1...: Various physics/drawing constants per entity type

Pic1SENT (Sprite ENTities) structure
16*16B: 16 entities sprites
    0: Player
    1: Standing log
    2: Tree standing log
    3: Horizontal log
    4: Vertical log
    5: Horizontal raft
    6: Vertical raft
    7: Snowman (alt: dark snowman)
    8: Sign (alt: unmarked sign)

Pic1STIL (Sprite TILes) structure
16B: Ground sprite
    0: Grass
3*16B: Miscellaneous
    0: Water
    1: Impenetrable Rock
    2: Stump
4*16B: 4 wall type sprites
    0: Wall
    1: Waterfall (wall but water)
    2: Wall with ladder
