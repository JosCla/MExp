L1 structure (768B total):
0-99: Various non-const static variables
    0-7: Files (EM, CE, EP, PD)
    8-9: Map width (MW)
    10-11: Map height (MH)
    12-17: Camera position (CX, CY, CZ)
    18-99: Unused
100-767: Stack (S is initially L1+768)

Str1F....: File strings

GDB1MEAH: Max entity ascension height
nB: The max height that each entity collision type can ascend in movement

Pic1SENT (Sprite ENTities) structure
16*16B: 16 entities sprites
    0: Player
    1: Standing log
    2: Tree standing log
    3: Horizontal log
    4: Vertical log
    5: Horizontal raft
    6: Vertical raft
    7: Snowman (alt: dark snowman)
    8: Sign (alt: unmarked sign)
    9: Postbox (alt: dark postbox)
    10: Alt-Snowman
    11: Alt-Sign
    12: Alt-Postbox
    13: UNUSED
    14: Snowman icon
    15: Island icon

Pic1STIL (Sprite TILes) structure
4*16B: 4 tile type sprites
    0: Grass
    1: Rock/Sand
    2-3: UNUSED
4*16B: 4 wall type sprites
    0: Wall
    1: Waterfall (wall but water)
    2: Wall with ladder
    3: Waterfall with ladder
8*16B: Miscellaneous
    0: Water
    1: Impenetrable Rock
    2: Stump
