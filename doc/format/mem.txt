L1 structure (768B total):
0-99: Various non-const static variables
    0-7: Files (EM, CE, EP, PD)
    8-9: Map width (MW)
    10-11: Map height (MH)
    12-17: Camera position (CX, CY, CZ)
    18-99: Unused
100-767: Stack (S is initially L1+768)

Str1F....: File strings

GDB1MEAH: Max entity ascension height
nB: The max height that each entity collision type can ascend in movement
