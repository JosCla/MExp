MExp Entity Types

0: Player
1: Standing log
2: Tree Standing log
3: Horizontal log
4: Vertical log
5: Horizontal raft
    B1-2: ID of other log in raft
6: Vertical raft
    B1-2: ID of other log in raft
7: Snowman
    B1: Snowman number
8: Sign
    B1: Sign number
9: Postbox
    B1: Postbox number

Entity Collision Types

0: Player
1: Standing log
2: Tree Standing log
3: Rolling log
4: Vertical log
5: Raft
6: Immobile

Entity Interaction Actions

0: Stop
1: Move (will only move if is top entity)
2: Push
3: Push then move (only moves in if push successful)
4: Push then climb (will move on top if push fails and target is top entity)
5: Move twice (Standing log rolls over rolling log)

Entity Movement Results

0: Success
1: Out of Bounds (player trying to move out of bounds)
2: Blocked (player bumping into wall, pushing blocked entity, etc)
3: Couldn't move (player trying to move into water, off sideways log, etc)
